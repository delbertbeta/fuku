# Change: Add outfit calendar tab

## Why

Users need a calendar view to plan and review outfit usage by date, making it easier to track what they wore across the month.

## What Changes

- Add a new "日历" tab that opens a month-view calendar.
- Allow associating multiple existing outfits with a date and viewing them per day.
- Enable add/remove actions for date-outfit associations and show details for the selected date.

## Impact

- Affected specs: ui, responsive-interface, outfit-management, calendar-management (new)
- Affected code: navigation UI, calendar page components, outfit APIs, database schema
